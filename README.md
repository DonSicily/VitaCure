# 🌿 VitaCure - Natural Medication Marketplace

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/vitacure)

## Vision
Return to natural medicine. **Health is Wealth.** VitaCure connects users with 100% natural remedies, ranked by **Active Percentage** - a unique algorithm that prioritizes potency over popularity.

## Core Features
- 🔍 **Smart Search**: Find remedies by ailment (insomnia, joint pain, anxiety)
- 📊 **Active Percentage™**: Filter products by concentration of active compounds
- 🌍 **Global Delivery**: Ship natural medicine worldwide
- 📖 **Remedy Profiles**: Pop-up displays with origin stories & full ingredients
- 🛒 **Seamless Cart**: Easy checkout with international shipping

## Tech Stack
- **Frontend**: React + Vite + TailwindCSS (PWA)
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Deployment**: Railway
